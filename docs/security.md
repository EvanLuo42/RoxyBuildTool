# Security policy

RoxyBuildTool has no stable release. Security fixes are applied to the latest revision on `master`; prerelease packages are supported on a best-effort basis.

Do not report suspected vulnerabilities in a public issue. Use [GitHub private vulnerability reporting](https://github.com/EvanLuo42/RoxyBuildTool/security/advisories/new).

Include the affected commit or package version, impact, reproduction steps, and any known mitigation. Do not include credentials, proprietary SDK material, or other secrets in the report.
